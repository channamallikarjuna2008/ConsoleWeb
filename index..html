About Visual Studio Code

> Visual Studio Code is another browser where you can code. It is just like Visual Studio (The main browser to code websites) except it has a few different features. At its heart, Visual Studio Code features a lightning fast source code editor, perfect for day-to-day use. With support for hundreds of languages, VS Code (Visual Studio Code) helps you be instantly productive with syntax highlighting, bracket-matching, auto-indentation, box-selection, snippets, and more. Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, Mac0S and Linux. It comes with built-in support for JavaScript, TypeScript and Node. Thanks for reading a little about Visual Studio Code!

About Console

> Console is a basic computer or monitor and keyboard that is connected to another computer, server, or a mainframe over a network. It is used to maintain or monitor the status of the network or computer. A console is the combination of a monitor and keyboard. It is a rudimentary interface in which the monitor provides the output and the keyboard is used for input. These consoles typically include a mouse for navigating graphical interfaces. ... The terms "console" and "terminal" are often used synonymously. Thanks for reading about console!
